Introduction

Network Security Wiki is a free encyclopedia that provides in-depth information about relevant topics in network security. This encyclopedia was created by Patrick Louis Aldover.

About The Author

Patrick Louis Aldover is a Cyber Security Master student at ETH Zürich. He previously finished his Bachelor at Graz University of Technology with honors. For his Bachelor thesis titled "Universal Biometric Authentication" he implemented a FIDO2-based prototype that enables clients to log into web services using Android devices as authentication tokens.
